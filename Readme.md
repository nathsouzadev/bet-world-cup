## Run bet-world-cup-api

### Installation

```bash
$ yarn install
```

### Running the app

```bash
# development
$ yarn start:dev

# build
$ yarn start:build

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test
```

### Local configuration
```bash
# base_url
http://localhost:5000/
```
